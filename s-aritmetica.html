Progressão Aritmética (PA)

O que é uma Progressão Aritmética?

Uma Progressão Aritmética (PA) é uma sequência numérica em que cada termo, a partir do segundo, é obtido somando-se uma constante ao termo anterior. Essa constante é chamada de razão (r).

Fórmulas Importantes

Termo Geral:
aₙ = a₁ + (n - 1) × r
Onde:
• aₙ = termo que queremos encontrar
• a₁ = primeiro termo
• r = razão
• n = posição do termo

Soma dos n primeiros termos:
Sₙ = n × (a₁ + aₙ) / 2
ou
Sₙ = n × [2a₁ + (n-1) × r] / 2

Exemplos

Exemplo 1: PA (5, 8, 11, 14, 17, ...)
• Primeiro termo: a₁ = 5
• Razão: r = 3 (8 - 5 = 3)
• 10º termo: a₁₀ = 5 + (10-1) × 3 = 5 + 27 = 32

Exemplo 2: PA (20, 15, 10, 5, 0, ...)
• Primeiro termo: a₁ = 20
• Razão: r = -5 (15 - 20 = -5)
• Esta é uma PA decrescente

Tipos de PA

PA Crescente: r > 0 (os termos aumentam)
Exemplo: 2, 5, 8, 11, 14...

PA Decrescente: r < 0 (os termos diminuem)
Exemplo: 100, 90, 80, 70, 60...

PA Constante: r = 0 (todos os termos são iguais)
Exemplo: 7, 7, 7, 7, 7...

Propriedades

• Em uma PA, qualquer termo (exceto os extremos) é a média aritmética dos termos vizinhos.

• A soma de termos equidistantes dos extremos é constante e igual à soma dos extremos.

• Se três números estão em PA, o termo do meio é chamado de média aritmética dos outros dois.
