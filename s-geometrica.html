O que é uma Progressão Geométrica?

Uma Progressão Geométrica (PG) é uma sequência numérica em que cada termo, a partir do segundo, é obtido multiplicando o termo anterior por uma constante chamada razão (q).

Fórmulas Importantes

Termo Geral:
aₙ = a₁ × q^(n-1)
Onde:
• aₙ = termo que queremos encontrar
• a₁ = primeiro termo
• q = razão
• n = posição do termo

Soma dos n primeiros termos:
Quando q ≠ 1: Sₙ = a₁ × (q^n - 1) / (q - 1)
Quando q = 1: Sₙ = n × a₁

Exemplos

Exemplo 1: PG (2, 6, 18, 54, ...)
• Primeiro termo: a₁ = 2
• Razão: q = 3 (6/2 = 3)
• 5º termo: a₅ = 2 × 3⁴ = 2 × 81 = 162

Exemplo 2: PG (100, 50, 25, 12.5, ...)
• Primeiro termo: a₁ = 100
• Razão: q = 0.5 (50/100 = 0.5)
• Esta é uma PG decrescente

Tipos de PG

PG Crescente: q > 1 (os termos aumentam)
Exemplo: 2, 4, 8, 16, 32...

PG Decrescente: 0 < q < 1 (os termos diminuem)
Exemplo: 100, 50, 25, 12.5...

PG Constante: q = 1 (todos os termos são iguais)
Exemplo: 5, 5, 5, 5...

PG Oscilante: q < 0 (os termos alternam sinais)
Exemplo: 3, -6, 12, -24...
